
# CV Accesorios - Ecommerce Setup

Este proyecto fue desarrollado con React, TypeScript y Tailwind CSS.

## Pasos para ejecutar en tu PC:

1. **Instalar dependencias:**
   ```bash
   npm install
   ```

2. **Ejecutar en modo desarrollo:**
   ```bash
   npm run dev
   ```

3. **Estructura de archivos sugerida:**
   - `src/types.ts`: Definiciones de interfaces.
   - `src/constants.tsx`: Datos de productos y configuración (WhatsApp, Email).
   - `src/components/`: Componentes reutilizables (Carrito, Cards).
   - `src/App.tsx`: Lógica principal y secciones.

## Personalización:
Para cambiar los productos, edita el array `PRODUCTS` en `src/constants.tsx`.
Para cambiar los datos de contacto, edita `WHATSAPP_NUMBER` y `CONTACT_EMAIL` en el mismo archivo.
